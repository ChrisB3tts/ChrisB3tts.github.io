Project 4

Air Line Satisfaction

Introduction

What factors affect customer satisfaction with airlines?

This project is aimed to use clustering to determine which factors affect customer satisfaction with airlines.

What is clustering and how does it work.

Clustering is a technique used to group similar data points together. It is a type of unsupervised learning. It works by finding clusters of data points that are similar to each other.

What is unsupervised learning?

Unsupervised learning is a type of machine learning where the model learns from the data without any predefined labels.
