Project 3

House Price Prediction

Introduction

Can the price of a house be predicted based on the features of the house?

This project is aimed to accurately predict the selling price of houses using the characteristics of the houses by using a continuous regression model.

What is a continuous regression model?

A continuous regression model is a type of regression analysis that is used to predict a continuous outcome variable. It is also known as a linear regression model. In a continuous regression model, the outcome variable is a continuous variable, such as a numerical value or a percentage.

Data Source

House Sales Prediction

Preprocessing

To deal with the Null/NaN I had to change the string "NaN" into Numpy "nan". A lot of the "NaN" data wasn't missing data, but it means not applicable. I was originally using pandas' get_dummies function, but I later switched to One Hot encoding as it is more sophisticated and leads to less multicollinearity.

Experiment 1

My first model was basic linear regression, and it scored a 1.20574. The R^2 is decent on this model at 0.81895.

Experiment 2

My second model used Lasso regression, it scored a 1.00646. Lasso regression was a little better than basic linear regression. Lasso has the ability to perform feature selection that forces less important coefficients to zero.

Experiment 3

My third model was gradient boosted regression which scored 0.20988. Gradient boosted regression is powerful and accurate. It can capture non-linear relationships, interactions between variables, and handle outliers and missing values automatically.

Conclusion

The price of a house can be predicted using a continuous regression model. The best model was gradient boosted regression. If I were to do things differently, I would use a hyperparameter optimization algorithm to find the best hyperparameters for the model. Looking at the leaderboard, people have gotten it down 0.00044 which has an almost perfect accuracy.

Impact

This project could be used in both a positive and negative way. On one hand it can help a customer looking for a new home to find a fair price for a house. On the other hand it can be used by realtors to artificially increase the price to above what the property is actually worth by only using the empirical values due to homes being more than just numbers in a spread sheet.
